How to Use Map File

  1. Press WIN + R and type %AppData% in the box
  2. Go to the folder .minecraft
  3. Then navigate to the saves folder (Place unzipped map folders here to play them in Minecraft)

How to Use the Modpack

  1. Open CurseForge launcher
  2. Ensure you're on the Minecraft tab, then click Create Custom Profile
  3. Click the small text saying Import and attach the Aincraft.zip file

Closing Notes and common issues

When putting the map folder into the saves directory, ensure:

  • The file is not a zip file
  • The file structure is correct (When extracting zip folders, it might sometimes put all the contents in a folder. Ensure only one folder is placed in the saves directory)
